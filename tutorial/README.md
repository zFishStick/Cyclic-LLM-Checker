# Introductory Notebooks
This folder contains introductory notebooks that provide some basic example about the usage of the selected tools used for our methodology.

The selected tools are:
- [Matplotlib](https://matplotlib.org/)
- [Pandas](https://pandas.pydata.org/)
- [Gemini API](https://ai.google.dev/)
- [DeepSeek API](https://api-docs.deepseek.com/)

There are two notebooks:
- `introductory_notebook_1.ipynb`: This notebook provides a basic introduction to the libraries `Matplotlib` and `Pandas`, showcasing some fundamental functionalities and how to use them.
- `introductory_notebook_2.ipynb`: This notebook focuses on the usage of the `Gemini API` and `DeepSeek API`, demonstrating how to interact with these APIs and utilize their features.
